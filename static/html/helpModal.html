Welcome to LEML Editor!

Using the canvas:

Select an element:

Click an element to select it (the border will become red)

Add an element:

To add an element, drag and drop it from the toolbar below the canvas

Delete an element:

To delete an element, select the element and then right click it

Add an action:

To add an action, select an element (which will become the arrow tail) and then right click another element (which will be the arrow head)
