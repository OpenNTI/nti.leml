Welcome to LEML Editor!

Using the canvas:

Select an element:

Click an element to select it (the border will become red)

Add an element:

To add an element from the toolbar below the canvas

  • Drag and drop it anywhere on the canvas
    • or
  • Double click it (adds it to the center of canvas)

Delete an element:

To delete an element, select the element and then

  • Right click it
    • or
  • Press Delete on the keyboard
    • or
  • Press Backspace on the keyboard

Add an action:

To add an action, select an element (which will become the arrow tail) and then

  • Right click another element (which will be the arrow head)
    • or
  • Shift-click another element (which will be the arrow head)

Add a building block to a context:

Select the context you want the building block to be in

  • Right click the building block to add it to the context
    • or
  • Shift-click the building block to add it to the context
